# 描画要素

SkiaSharpを使用して、以下の項目を表示するシステムモニタの画面をoutput.pngとして出力する。
画像サイズは1280x480とする。
ハイクオリティなデザインにしてください。

## CPU

- CPU使用率(%)
- CPU使用率内訳:システム(%)
- CPU使用率内訳:ユーザー(%)
- CPU使用率内訳:アイドル(%)
- CPU Eコア使用率(%)
- CPU Pコア使用率(%)
- ロードアベレージ1分
- ロードアベレージ5分
- ロードアベレージ15分
- CPU周波数(MHz)
- CPU Eコア周波数(MHz)
- CPU Pコア周波数(MHz)

## メモリ

- 使用率(%)
- 使用済みメモリ(GB)
- アプリケーションメモリ(GB)
- 書き干されているメモリ(GB)
- スワップ使用領域(MB)

## ディスク

- 使用率(%)、空きGB / 容量GB
- 読み込み量 KB/s
- 書き込み量 LB/s

## ネットワーク

- 転送量 DL方向 MB/s
- 転送量 UP方向 MB/s

## プロセス

- プロセス数合計
- スレッド数合計

## HWモニター

- CPU温度(℃)
- GPU温度(℃)
- 電力システムトータル(W)
- 電力:CPU(W)
- 電力:GPU(W)

## システム

- 稼働時間

## 方針

- 項目の重なりがないようにする
- タイムライン以外では同じ項目は表示しない
- 画面内に全ての項目が表示されるようにする
- 各項目は小さなWidgetとして配置し、将来的に項目の増加ができるようにする
- CPU関連の項目を1つのWidgetにするのではなく、CPU使用率(%)、CPU使用率内訳:システム(%)等の単位でWidgetにする
- 画面上部または画面左に、タイトル、Uptime、現在時刻のようなものを共通的に表示する
- パディング、ボーダーは少なめで余白があまりすぎないデザインにする
- Widget内に項目名と値を左右に配置する場合、項目名が長くなりすぎるなら2段で描画する
- CPU、Memory等は円ゲージで表示する、ディスク使用量ばバーゲージで描画する
- 1280x480といっても6.86インチの出力であることを想定したフォントサイズにしてください
- 4x4のグリッドをベースにWidgetを配置してください
- 全ての項目を出力しなくても良いです、優先度が高いと考えるものを出力してください
