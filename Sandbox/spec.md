# 描画要素

SkiaSharpを使用して、以下の項目を表示するシステムモニタの画面をoutput.pngとして作成する。
画像サイズは1280x480とする。
SF的な雰囲気でハイクオリティなものとする。

## CPU

- CPU使用率(%)
- CPU使用率内訳:システム(%)
- CPU使用率内訳:ユーザー(%)
- CPU使用率内訳:アイドル(%)
- CPU Eコア使用率(%)
- CPU Pコア使用率(%)
- ロードアベレージ1分
- ロードアベレージ5分
- ロードアベレージ15分
- CPU周波数(MHz)
- CPU Eコア周波数(MHz)
- CPU Pコア周波数(MHz)

## メモリ

- 使用率(%)
- 使用済みメモリ(GB)
- アプリケーションメモリ(GB)
- 書き干されているメモリ(GB)
- スワップ使用領域(MB)

## ディスク

- 使用率(%)、空きGB / 容量GB
- 読み込み量 KB/s
- 書き込み量 LB/s

## ネットワーク

- 転送量 DL方向 MB/s
- 転送量 UP方向 MB/s

## プロセス

- プロセス数合計
- スレッド数合計

## HWモニター

- CPU温度(℃)
- GPU温度(℃)
- 電力システムトータル(W)
- 電力:CPU(W)
- 電力:GPU(W)

## システム

- 稼働時間
